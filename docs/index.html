dicto

音声・動画をテキストに。完全ローカル処理。

🔒
データは外部に送信されません
🖱️
ドラッグ&ドロップで簡単操作
🌐
ネット不要で動作
📝
タイムスタンプ付き出力

対応形式: mp3, wav, m4a, mp4, mov ほか

インストール

1 ターミナルを開く

Launchpad → 「ターミナル」で検索してクリック
（または Spotlight: Cmd+Space → 「ターミナル」と入力）

2 以下をコピーして貼り付け、Enterキーを押す
curl -fsSL https://ykshio.github.io/dicto/install.sh | bash
3 待つだけ（10〜20分）

自動でインストールされます。
途中でパスワードを聞かれたらMacのログインパスワードを入力してください。

4 完了！

Launchpad に「dicto」アプリが追加されます。
クリックするだけで起動します。

⚠️ 初回起動時

「開発元が未確認」と表示されたら、アプリを右クリック →「開く」で起動できます。

アップデート

同じコマンドを再実行するだけで最新版に更新されます。

curl -fsSL https://ykshio.github.io/dicto/install.sh | bash
