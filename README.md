# 🎙️ 文字起こしアプリ 使い方

音声・動画ファイルから文字を書き起こすアプリです。
インターネットに送信せず、すべてこのMacの中で処理されます。

---

## 📦 初回セットアップ(最初の1回だけ)

1. このフォルダをMacのわかりやすい場所(書類フォルダなど)に置く
2. フォルダ内の **`install.command`** をダブルクリック
3. 「開発元が未確認」というメッセージが出たら:
   - システム設定 → プライバシーとセキュリティ → 下の方の「このまま開く」
   - または、ファイルを右クリック → 「開く」
4. ターミナルウィンドウが開いたら、画面の指示に従う
   - 途中でパスワードを求められたら、Macのログインパスワードを入力
   - 完了まで10〜20分ほどかかります
5. 「インストール完了!」が表示されたら閉じてOK

---

## ▶️ 使い方(2回目以降)

1. フォルダ内の **`起動.command`** をダブルクリック
2. 自動的にブラウザが開きます(1〜2分お待ちください)
3. 音声ファイルや動画ファイルを画面にドラッグ&ドロップ
4. 「文字起こしを開始」ボタンを押す
5. 完了すると、結果が画面に表示され、**デスクトップにテキストファイルが保存**されます
6. 使い終わったら、ターミナルのウィンドウを閉じる(またはCtrl+Cを押す)

---

## 📝 対応ファイル形式

- 音声: mp3, wav, m4a, aac, flac, ogg
- 動画: mp4, mov, avi, mkv, wmv
- その他ffmpegが対応する形式

---

## ⏱️ 処理にかかる時間の目安

| 音声の長さ | おおよその処理時間 |
|---|---|
| 5分 | 1〜3分 |
| 30分 | 5〜15分 |
| 1時間 | 10〜30分 |

※ Macの性能やモデル設定によって変わります

---

## ❓ 困ったときは

**Q. インストールが失敗する**  
A. Wi-Fiに繋がっているか確認してください。もう一度 install.command を実行してみてください。

**Q. 「開発元が未確認」で開けない**  
A. ファイルを右クリック → 「開く」を選んでください。

**Q. 精度を上げたい / もっと早くしたい**  
A. `app.py` の `MODEL_SIZE` を変更してください:
   - 速度重視: `"small"`
   - バランス: `"medium"` (デフォルト)
   - 精度重視: `"large-v3"` (遅いが高精度)

**Q. アプリを終了するには?**  
A. ターミナルウィンドウで Ctrl+C を押すか、ウィンドウを閉じてください。

---

## 🔒 プライバシーについて

このアプリはすべてMacの中だけで処理します。
音声データがインターネットに送信されることはありません。
