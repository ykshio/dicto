# 🎙️ dicto - 文字起こしアプリ

音声・動画ファイルから文字を書き起こすアプリです。
インターネットに送信せず、すべてこのMacの中で処理されます。

---

## 📦 インストール

ターミナルで以下を実行してください（初回のみ）:

```
curl -fsSL https://ykshio.github.io/dicto/install.sh | bash
```

完了まで10〜20分ほどかかります。途中でパスワードを求められたら、Macのログインパスワードを入力してください。

---

## ▶️ 使い方

1. Launchpad から **dicto** を起動
2. 自動的にブラウザが開きます（1〜2分お待ちください）
3. 音声ファイルや動画ファイルを画面にドラッグ&ドロップ
4. 「文字起こしを開始」ボタンを押す
5. 完了すると、結果が画面に表示され、**デスクトップにテキストファイルが保存**されます

---

## 📝 対応ファイル形式

- 音声: mp3, wav, m4a, aac, flac, ogg
- 動画: mp4, mov, avi, mkv, wmv
- その他ffmpegが対応する形式

---

## ⏱️ 処理にかかる時間の目安

| 音声の長さ | おおよその処理時間 |
|---|---|
| 5分 | 1〜3分 |
| 30分 | 5〜15分 |
| 1時間 | 10〜30分 |

※ Macの性能やモデル設定によって変わります

---

## ❓ 困ったときは

**Q. インストールが失敗する**
A. Wi-Fiに繋がっているか確認してください。もう一度同じコマンドを実行してみてください。

**Q. 「開発元が未確認」で開けない**
A. アプリを右クリック → 「開く」を選んでください。

**Q. 精度を上げたい / もっと早くしたい**
A. アプリ画面の「設定」からモデルサイズを変更できます:
   - 速度重視: `small`
   - バランス: `medium`（デフォルト）
   - 精度重視: `large-v3`（遅いが高精度）

**Q. アップデートするには?**
A. インストールと同じコマンドを再実行するだけで最新版に更新されます。

---

## 🔒 プライバシーについて

このアプリはすべてMacの中だけで処理します。
音声データがインターネットに送信されることはありません。
